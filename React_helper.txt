on windows OS

create a project :
install node.js
confirm add to path 
open cmd on spcific folder
type: npx create-react-app my_app_name (my_app_name = must be lower case)
hit enter and your app is create

if you couldn't able to find "src" folder then and get messege after compeletion of process:
messege:
	"A template was not provided. This is likely because you're using an outdated version of create-react-app. Please note that global installs of create-react-app are no longer supported."

then
type: npm uninstall -g create-react-app

	to ensure that "npx" use latest vrsion of "create-react-app"


if we are createing any custom component then name of that component must start with capital letter.
